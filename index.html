人脉雷达 v2.1

你的专属人脉秘书 Agent。说一段话，AI秘书帮你记住、分析、连接一切人脉。

核心理念：每次语音录入 = 给你的 Agent 喂数据。你的人脉秘书吸收足够信息后，会主动推送高价值洞察；连接好友后，你的秘书 × 他的秘书自动交互，挖掘双方人脉中的合作机会。
5层
Agent分析深度
1000万
架构目标用户

Agent 的两大工作模式

  • 模式一：智能推送 — Agent 吸收你的人脉数据后，从供需链/资源互补/信息差/社交货币/时机 5层分析，主动发现你看不到的机会
  • 模式二：秘书×秘书 — 连接好友后，两个Agent自动交互，匹配双方人脉中的互补资源和合作可能

产品特点

  • 唯一操作：说一段话，Agent完成一切（录入/分析/匹配/推送）
  • AI生成完整备注摘要，不丢失任何细节
  • 每条推送含可直接使用的行动话术
  • 清晰文字标签，对所有年龄段友好

技术架构

  • iOS: SwiftUI + Apple Speech (中文语音)
  • 后端: FastAPI + PostgreSQL 16 + Redis 7
  • AI: Claude Sonnet (结构化 + 5层匹配引擎)
  • 千万用户：连接池 + Nginx LB + 水平扩展
